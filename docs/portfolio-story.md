# Portfolio Story

Open IFC Viewer Case Study is designed to show practical front-end and BIM visualization skills in a safe public format.

It highlights:

- TypeScript and React application structure
- Three.js scene management
- Local IFC file processing in the browser
- Viewer state handling and object inspection
- A dashboard-oriented UI suitable for portfolio presentation

The current scope is intentionally focused and leaves room for future expansion such as model trees, sectioning, saved viewpoints, filtering, and large-model optimization.
